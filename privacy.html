Política de Privacidade – AutoFlow

Última atualização: 13/12/2025

A extensão AutoFlow foi desenvolvida com o objetivo exclusivo de permitir que o usuário grave e reproduza ações simples no navegador, como cliques e preenchimento de campos. Esta política explica como a extensão trata os dados do usuário.

1. Coleta de Dados

A extensão AutoFlow não coleta, não armazena, não transmite e não compartilha nenhum dado pessoal, sensível ou identificável do usuário.

Nenhum dos seguintes dados é coletado:

  • Nome, e-mail, telefone ou outras informações pessoais
  • Dados financeiros ou informações de pagamento
  • Histórico de navegação
  • Senhas ou dados de autenticação
  • Localização ou endereço IP
  • Conteúdo de sites visitados
  • Informações de uso do navegador

2. Ações Gravadas

As ações gravadas pela extensão (como cliques ou valores digitados) são armazenadas somente no dispositivo do usuário, utilizando o armazenamento local do navegador (chrome.storage.local).

Essas informações:

  • são usadas exclusivamente para executar a automação solicitada
  • nunca são enviadas para servidores externos
  • podem ser apagadas a qualquer momento pelo próprio usuário

3. Permissões Utilizadas

A extensão utiliza as permissões:

  • scripting – para injetar scripts necessários à gravação e execução
  • tabs e activeTab – para atuar apenas na aba autorizada
  • storage – para salvar ações localmente

Nenhuma dessas permissões é usada para coletar ou transmitir dados.

4. Compartilhamento de Dados

A extensão AutoFlow não compartilha nenhum tipo de dado com terceiros.

5. Segurança

Todo o processamento ocorre localmente no navegador. Não há comunicação com servidores externos.

6. Alterações a Esta Política

Esta política pode ser atualizada conforme necessário. O usuário será informado caso ocorram mudanças relevantes.
